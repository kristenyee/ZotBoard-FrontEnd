How to Use ZotBoard:

Use the buttons below to create your board.

  • Use the "Add Text" button to type. Double click the textbox to edit.
  • Use the "Draw" button to use the pencil. Use the "Stop Drawing" button to switch back to a cursor.
  • Use the insert shape buttons to add any shapes you need.
  • Click an item to change its size and move it.
  • To delete items, select them and drag them off the board.
